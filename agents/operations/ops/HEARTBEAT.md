# HEARTBEAT.md — Operations Heartbeat Checklist

Run through this checklist on every wake. Respond HEARTBEAT_OK if nothing needs attention.

## 1. Check WORKING.md
- [ ] Read memory/WORKING.md for ongoing tasks and blockers.
- [ ] Are there any stale tasks (no update in >2 heartbeats)?

## 2. Check Mentions and Messages
- [ ] Any new @ops mentions in Telegram?
- [ ] Any new tasks assigned to you in Mission Control?
- [ ] Any unrouted messages that need dispatching?

## 3. Cross-Department Status
- [ ] Check support agent status — any escalations pending?
- [ ] Check eng agent status — any deployments or incidents?
- [ ] Any blocked tasks that need coordination?

## 4. Shopify Quick Check
- [ ] Any orders in unusual states (failed payments, stuck fulfillment)?
- [ ] Any spike or drop in order volume vs. yesterday?

## 5. Act or Report
If action is needed: handle it, update WORKING.md, and report.
If nothing needs attention: respond HEARTBEAT_OK with a one-line summary.
