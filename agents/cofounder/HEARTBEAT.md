# HEARTBEAT.md — Cofounder Heartbeat Checklist

Run through this checklist on every wake. Respond HEARTBEAT_OK if nothing needs attention.

## 1. Check WORKING.md
- [ ] Read memory/WORKING.md for active opportunities and strategic questions
- [ ] Any decisions pending for >48 hours?
- [ ] Any competitive intel that needs updating?

## 2. Market Scan
- [ ] Any industry news or trends worth noting?
- [ ] Any competitor moves to analyze?
- [ ] Any new opportunities emerging?

## 3. Cross-Department Insights
- [ ] Check Support — any recurring customer pain points?
- [ ] Check Engineering — any technical constraints or opportunities?
- [ ] Check Ops — any operational bottlenecks or efficiencies?

## 4. Strategic Questions
- [ ] Are we building toward "best in the world"?
- [ ] What's our biggest risk? What's our biggest opportunity?
- [ ] What should we stop doing?

## 5. Act or Report
If insights or recommendations emerge: document them, tag relevant agents, and report.
If nothing needs attention: respond HEARTBEAT_OK with a one-line summary.
