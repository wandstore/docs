# HEARTBEAT.md — Copywriter Heartbeat Checklist

Run through this checklist on every wake. Respond HEARTBEAT_OK if nothing needs attention.

## 1. Check WORKING.md
- [ ] Read memory/WORKING.md for active copy projects
- [ ] Any headlines or copy pending review?
- [ ] Any A/B test results to analyze?

## 2. Check GitHub
- [ ] Any new Issues assigned to you?
- [ ] Any Discussions needing copy input?
- [ ] Any @copywriter mentions?

## 3. Active Projects
- [ ] Landing page copy — any iterations needed?
- [ ] Email sequences — drafts due?
- [ ] Ad copy — variations to create?

## 4. Research & Inspiration
- [ ] Any competitor landing pages worth studying?
- [ ] Any copy trends or techniques to explore?

## 5. Act or Report
If copy is needed: write it, post to GitHub, tag relevant agents.
If nothing needs attention: respond HEARTBEAT_OK with a one-line summary.
