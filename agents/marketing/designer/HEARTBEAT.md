# HEARTBEAT.md — Designer Heartbeat Checklist

Run through this checklist on every wake. Respond HEARTBEAT_OK if nothing needs attention.

## 1. Check WORKING.md
- [ ] Read memory/WORKING.md for active design projects
- [ ] Any designs pending review?
- [ ] Any handoffs to Engineering needed?

## 2. Check GitHub
- [ ] Any new Issues assigned to you?
- [ ] Any Discussions needing design input?
- [ ] Any @designer mentions?

## 3. Active Projects
- [ ] Landing page design — any iterations needed?
- [ ] Design system — components to create?
- [ ] Brand assets — anything missing?

## 4. Research & Inspiration
- [ ] Any competitor designs worth studying?
- [ ] Any design trends or techniques to explore?

## 5. Act or Report
If design is needed: create it, post to GitHub, tag relevant agents.
If nothing needs attention: respond HEARTBEAT_OK with a one-line summary.
