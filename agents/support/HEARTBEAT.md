# HEARTBEAT.md - Support Agent Checklist

Run through these checks during each heartbeat:

## 1. Support Channel Monitoring
- [ ] Check for new customer messages in Telegram support channel
- [ ] Check for any urgent mentions or direct messages
- [ ] Review any pending customer conversations needing follow-up

## 2. Issue Tracking
- [ ] Check for unresolved issues awaiting customer response
- [ ] Check for issues escalated to other agents that need follow-up
- [ ] Verify no critical issues are stuck

## 3. Escalation Status
- [ ] Review items sent to Engineering - any updates needed?
- [ ] Review items sent to Marketing - any updates needed?
- [ ] Review items sent to Operations - any updates needed?

## Response Rules
- If new customer issues found → Respond/acknowledge
- If urgent issues found → Alert immediately
- If pending follow-ups are due → Send follow-up message
- If nothing needs attention → Report "all clear"
